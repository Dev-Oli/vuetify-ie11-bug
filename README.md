# Readme
A sample repository for reproducing the vuetify problems on IE11.

## Dependencies
* Node version: 12.10.0
* npm version: 6.14.5
* @vue/cli version: 4.3.1

## Project setup
```
vue create ie-test // -> select default option

cd ie-test

vue add vuetify // -> select default option

npm run serve
```
